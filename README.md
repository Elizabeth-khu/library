## Overview

**Library** is a simple console-based CRUD application for managing books.  
The project is designed as a test assignment with a strong focus on **Clean Code**, **SOLID principles** (especially Single Responsibility Principle), and **Dependency Injection** using **Spring Context** (without Spring Boot).

The application allows users to manage a list of books stored in a CSV file via an interactive console menu.

---

## Features

- Display a list of books
- Create a new book
- Edit an existing book
- Delete a book
- Persistent storage using CSV
- Console-based user interface
- Clear separation of responsibilities
- Spring-based dependency injection (no Spring Boot)

---

## Tech Stack

- **Java 17**
- **Gradle**
- **Spring Context**
- **Jackson CSV (CsvMapper)**
